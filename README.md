## Install

```bash
$ npm i gupiao -g
```
## Usage

```js
Usage: index [options]

  Options:

    -V, --version  output the version number
    -r, --rocket   盘中异动实时监控
    --about        关于
    -h, --help     output usage information

```
##example
```js
$ gupiao -r //查看股票行情异动情况
  
```
